= Monitorar a migração de volumes do ONTAP
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Além de monitorar a migração geral da SVM com o `vserver migrate show` comando, você pode monitorar o status de migração dos volumes que o SVM contém.

.Passos
. No cluster de destino, verifique o status da migração de volume:
+
[source, cli]
----
vserver migrate show-volume
----

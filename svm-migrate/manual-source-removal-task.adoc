= Remover manualmente a SVM ONTAP de origem após a migração do cliente.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Se você tiver realizado a migração do SVM com a limpeza de origem desativada, poderá remover o SVM de origem manualmente após a conclusão da transferência do cliente.

.Passos
. Verifique se o status está pronto para a limpeza da origem:
+
[source, cli]
----
vserver migrate show
----
. Limpe a fonte:
+
[source, cli]
----
vserver migrate source-cleanup -vserver <SVM_name>
----

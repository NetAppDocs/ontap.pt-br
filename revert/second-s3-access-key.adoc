= Remova as segundas chaves de acesso S3 antes de reverter do ONTAP 9.19.1
:allow-uri-read: 


[role="lead"]
Se você atualizou para ONTAP 9.19.1 e adicionou uma segunda chave de acesso S3 a qualquer usuário, essas chaves devem ser removidas antes que você possa reverter para uma versão anterior do ONTAP.

.Passos
. link:../s3-config/regenerate-access-keys.html#show-s3-access-keys["Mostrar chaves de acesso S3"] para determinar se foram adicionadas segundas chaves de acesso para algum usuário do S3.
. link:../s3-config/regenerate-access-keys.html#remove-an-s3-access-key["Remova as segundas chaves de acesso"] para todos os usuários do S3.

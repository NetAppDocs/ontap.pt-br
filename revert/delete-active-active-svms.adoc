= Exclua as SVMs ativo-ativo antes de reverter do ONTAP 9.19.1
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
O multipath ativo-ativo de SAN não é suportado no ONTAP 9.18.1 e versões anteriores. Antes de reverter do ONTAP 9.19.1, você deve excluir todas as máquinas virtuais de armazenamento (SVMs) com configurações ativo-ativo.

.Passos
. Identifique quaisquer SVMs que tenham a opção san-multipathing configurada para ativo-ativo ou ativo-ativo-scsi-only:
+
[source, cli]
----
vserver show
----
. Exclua quaisquer SVMs configuradas para multipath ativo-ativo ou multipath ativo-ativo-somente iSCSI:
+
[source, cli]
----
vserver delete -vserver <SVM>
----
. Verifique se as SVMs foram excluídas:
+
[source, cli]
----
vserver show
----

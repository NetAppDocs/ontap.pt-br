= Saiba mais sobre a atualização do ONTAP
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Ao atualizar seu software ONTAP, você pode aproveitar os novos e aprimorados recursos do ONTAP que ajudam a reduzir custos, acelerar workloads críticos, melhorar a segurança e expandir o escopo de proteção de dados disponível para sua organização.

Uma atualização principal do ONTAP consiste na migração de uma versão numerada inferior do ONTAP para uma superior. Um exemplo seria a atualização do seu cluster do ONTAP 9.14.1 para ONTAP 9.18.1. Uma atualização secundária (ou de correção) consiste na migração de uma versão do ONTAP para uma versão do ONTAP dentro da mesma numeração. Um exemplo seria a atualização do seu cluster do ONTAP 9.17.1P1 para 9.17.1P4.

Para começar, você pode usar um dos seguintes métodos para se preparar para a atualização.

* Upgrade Advisor
+
Se você possui um contrato ativo do SupportEdge para Active IQ Digital Advisor (também conhecido como Digital Advisor), você deve link:create-upgrade-plan.html["Prepare-se para atualizar com o Upgrade Advisor"]. O Upgrade Advisor fornece inteligência que ajuda a minimizar incertezas e riscos, avaliando seu cluster e criando um plano de atualização específico para sua configuração.

+
O Upgrade Advisor é adequado para usuários com ambientes menores que preferem uma experiência baseada em nuvem e centrada na interface do usuário.

+
Saiba mais sobre link:https://docs.netapp.com/us-en/active-iq/upgrade_advisor_overview.html["Upgrade Advisor"^].

* Verificador de integridade de upgrade
+
Você também pode link:create-upgrade-plan.html["prepare-se para atualizar com o Upgrade Health Checker"]. Upgrade Health Checker é uma ferramenta de serviço de consultoria de atualização que realiza verificações de integridade abrangentes no local para identificar bloqueadores e avisos de atualização. Um contrato SupportEdge não é necessário para usar o Upgrade Health Checker.

+
O Upgrade Health Checker é adequado para usuários que possuem um ambiente grande ou complexo, acesso limitado à internet ou estrutura de automação e orquestração existente.

+
Saiba mais sobre link:https://docs.netapp.com/us-en/upgrade-health-checker/learn-about-upgrade-health-checker.html["Verificador de integridade de upgrade"^].

* Preparação manual
+
Se você não utiliza o Upgrade Advisor ou a ferramenta Upgrade Health Checker, é necessário link:prepare.html["prepare-se para atualizar manualmente"]. A preparação manual exige a conclusão de várias tarefas para verificar quaisquer problemas ou bloqueios conhecidos e garantir que seu cluster atenda aos requisitos necessários para a atualização.

+
Saiba mais sobre link:prepare.html["preparação manual para upgrade"^].



Depois de se preparar para a atualização, é recomendável que você execute atualizações usando link:task_upgrade_andu_sm.html["Atualização automatizada e sem interrupções (ANDU) do System Manager"]o . O ANDU aproveita a tecnologia de failover de alta disponibilidade (HA) do ONTAP para garantir que os clusters continuem fornecendo dados sem interrupção durante a atualização.


NOTE: A partir do ONTAP 9.12.1, o System Manager está totalmente integrado ao NetApp Console.  Se o Console estiver configurado no seu sistema, você poderá atualizar através da página Sistemas.

Se você quiser obter assistência para atualizar seu software ONTAP, os Serviços profissionais da NetApp oferecem um link:https://www.netapp.com/pdf.html?item=/media/8144-sd-managed-upgrade-service.pdf["Serviço de atualização gerenciada"^]. Se estiver interessado em utilizar este serviço, contacte o seu representante de vendas da NetApp ou link:https://www.netapp.com/forms/sales-contact/["Envie o formulário de inquérito de vendas da NetApp"^]. O Serviço de Atualização gerenciada, bem como outros tipos de suporte de atualização, estão disponíveis para clientes link:https://www.netapp.com/pdf.html?item=/media/8845-supportedge-expert-service.pdf["Serviços da SupportEdge Expert"^]sem nenhum custo adicional.

= Erro de comunicação de rede
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensagem*
+
`Network communication error`

* *Causa*
+
A comunicação com uma fita remota em uma conexão de três vias NDMP falhou.

* *Ações corretivas*
+
Verifique a ligação de rede ao motor remoto.

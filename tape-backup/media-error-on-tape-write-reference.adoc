= Erro de Mídia na gravação da fita
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensagem*
+
`Media error on tape write`

* *Causa*
+
A fita usada para o backup está corrompida.

* *Ações corretivas*
+
Substitua a fita e tente novamente o trabalho de backup.

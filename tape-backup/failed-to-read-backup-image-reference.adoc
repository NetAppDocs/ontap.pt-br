= Falha ao ler a imagem de cópia de segurança
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensagem*
+
`Failed to read backup image`

* *Causa*
+
A fita está corrompida.

* *Ações corretivas*
+
Se a fita estiver corrompida, não será possível executar a operação de restauração.

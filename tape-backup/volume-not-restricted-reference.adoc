= Volume não restrito
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensagem*
+
`Volume not restricted`

* *Causa*
+
O volume de destino para o qual os dados estão sendo restaurados não é restrito.

* *Ações corretivas*
+
Restrinja o volume e tente novamente a operação de restauração.

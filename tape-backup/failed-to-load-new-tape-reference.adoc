:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensagem*
+
`Failed to load new tape`

* *Causa*
+
Erro na unidade de fita ou Mídia.

* *Ações corretivas*
+
Substitua a fita e tente novamente a operação.

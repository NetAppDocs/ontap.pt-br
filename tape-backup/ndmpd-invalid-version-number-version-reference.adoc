= ndmpd número de versão inválido: version_number "
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensagem*
+
`ndmpd invalid version number: version_number`

* *Causa*
+
A versão NDMP especificada não é suportada pelo sistema de storage.

* *Ações corretivas*
+
Especifique a versão 4 do NDMP.

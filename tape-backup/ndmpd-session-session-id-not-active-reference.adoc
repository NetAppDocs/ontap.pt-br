= Sessão ndmpd session_ID não ativa
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensagem*
+
`ndmpd session session_ID not active`

* *Causa*
+
A sessão NDMP pode não existir.

* *Ações corretivas*
+
Use o `ndmpd status` comando para exibir as sessões NDMP ativas.

= Trabalho cancelado devido ao encerramento
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensagem*
+
`Job aborted due to shutdown`

* *Causa*
+
O sistema de armazenamento está sendo reinicializado.

* *Ações corretivas*
+
Tente novamente o trabalho depois que o sistema de armazenamento for reiniciado.

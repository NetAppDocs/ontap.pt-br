= Volume off-line
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensagem*
+
`Volume offline`

* *Causa*
+
O volume que está sendo feito backup está offline.

* *Ações corretivas*
+
Coloque o volume on-line e tente novamente o backup.

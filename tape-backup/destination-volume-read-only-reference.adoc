= O volume de destino é somente leitura
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensagem*
+
`Destination volume is read-only`

* *Causa*
+
O caminho para o qual a operação de restauração é tentada é somente leitura.

* *Ações corretivas*
+
Tente restaurar os dados para um local diferente.

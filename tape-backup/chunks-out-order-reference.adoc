= Pedaços fora de ordem
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensagem*
+
`Chunks out of order`

* *Causa*
+
As fitas de backup não estão sendo restauradas na sequência correta.

* *Ações corretivas*
+
Repita a operação de restauração e carregue as fitas na sequência correta.

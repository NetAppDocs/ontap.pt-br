= Cabeçalho da imagem ausente ou corrompido
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensagem*
+
`Image header missing or corrupted`

* *Causa*
+
A fita não contém um backup SMTape válido.

* *Ações corretivas*
+
Tente novamente com uma fita contendo um backup válido.

= Falha na gravação da fita - erro de Mídia encontrado na nova fita
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensagem*
+
`Tape write failed - new tape encountered media error`

* *Causa*
+
A fita usada para o backup está corrompida.

* *Ações corretivas*
+
Substitua a fita e tente novamente o backup.

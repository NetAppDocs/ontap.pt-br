= Formato de bloco não suportado
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensagem*
+
`Chunk format not supported`

* *Causa*
+
A imagem de backup não é do SMTape.

* *Ações corretivas*
+
Se a imagem de backup não for do SMTape, tente novamente a operação com uma fita que tenha o backup do SMTape.

= Soma de verificação da imagem de cópia de segurança inválida
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensagem*
+
`Invalid backup image checksum`

* *Causa*
+
A fita está corrompida.

* *Ações corretivas*
+
Se a fita estiver corrompida, não será possível executar a operação de restauração.

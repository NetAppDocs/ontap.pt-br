= Falha ao criar instantâneo
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensagem*
+
`Failed to create snapshot`

* *Causa*
+
O volume já contém o número máximo de instantâneos.

* *Ações corretivas*
+
Elimine alguns instantâneos e, em seguida, tente novamente a operação de cópia de segurança.

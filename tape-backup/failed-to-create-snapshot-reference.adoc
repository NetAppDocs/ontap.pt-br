= Falha ao criar instantâneo
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensagem*
+
`Failed to create snapshot`

* *Causa*
+
O volume já contém o número máximo de cópias Snapshot.

* *Ações corretivas*
+
Exclua algumas cópias Snapshot e tente novamente a operação de backup.

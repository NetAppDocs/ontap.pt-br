= Caminho de volume inválido
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensagem*
+
`Invalid volume path`

* *Causa*
+
O volume especificado para a operação de backup ou restauração não foi encontrado.

* *Ações corretivas*
+
Tente novamente o trabalho com um caminho de volume e um nome de volume válidos.

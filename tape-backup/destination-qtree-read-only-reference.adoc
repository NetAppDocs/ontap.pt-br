= A qtree de destino é somente leitura
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensagem*
+
`Destination qtree is read-only`

* *Causa*
+
A qtree para a qual a restauração é tentada é somente leitura.

* *Ações corretivas*
+
Tente restaurar os dados para um local diferente.

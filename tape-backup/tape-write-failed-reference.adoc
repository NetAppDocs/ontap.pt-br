= Falha na gravação em fita
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Mensagem*
+
`Tape write failed`

* *Causa*
+
A fita usada para o backup está corrompida.

* *Ações corretivas*
+
Substitua a fita e tente novamente o trabalho de backup.

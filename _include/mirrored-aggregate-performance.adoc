NOTE: Recomenda-se manter pelo menos 20% de espaço livre para agregados espelhados para obter desempenho e disponibilidade de armazenamento ideais.  Embora a recomendação seja de 10% para agregados não espelhados, os 10% adicionais de espaço podem ser usados pelo sistema de arquivos para absorver alterações incrementais.  Alterações incrementais aumentam a utilização do espaço para agregados espelhados devido à arquitetura baseada em snapshots com redirecionamento na gravação do ONTAP.  O não cumprimento dessas boas práticas pode ter um impacto negativo no desempenho.

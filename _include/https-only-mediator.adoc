* Ao instalar ou atualizar para ONTAP Mediator 1.12 ou posterior, você tem a opção de instalar o ONTAP Mediator com suporte a iSCSI e HTTPS, ou somente com suporte a HTTPS. Se optar por instalar o ONTAP Mediator apenas com suporte a HTTPS, o pacote SCST não será instalado.
* Se você atualizar do ONTAP Mediator 1.11 ou anterior com o SCST instalado e escolher a opção somente HTTPS durante a atualização, o pacote SCST será desinstalado e a nova versão do ONTAP Mediator não conterá o pacote SCST.
* Se você planeja usar ONTAP Mediator para monitorar várias configurações de MetroCluster IP, o suporte depende do seu tipo de Mediator, versão do ONTAP e sistema operacional. Consulte link:https://docs.netapp.com/us-en/ontap-metrocluster/install-ip/concept_mediator_requirements.html#mediator-multiple-mccip["Utilize ONTAP Mediator para monitorar várias configurações de IP MetroCluster"^] para mais informações.
* A instalação do ONTAP Mediator com suporte apenas para HTTPS é compatível com os seguintes casos de uso:
+
** Você planeja usar apenas o ONTAP Mediator para monitorar seus clusters de sincronização ativa do SnapMirror.
** Você planeja configurar ONTAP Mediator com HTTPS para monitorar uma ou mais configurações de IP MetroCluster.


* ONTAP Mediator deve ser instalado com suporte para iSCSI e HTTPS no seguinte cenário:
+
** Você planeja configurar ONTAP Mediator com iSCSI para monitorar uma ou mais configurações de IP do MetroCluster.
